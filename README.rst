An index of wheels

Not Supported
=============
- PyPI releases without wheels
- non-PEP440 version numbers (?)
- Wheel v1.9 (PEP 491, still in draft)
- wheels with invalid or non-UTF-8 metadata
- large wheels
- PEP 561 (still in draft)
